# Portal Core ProEmpresa

Variable requerida en Vercel:

```env
VITE_API_BASE_URL=https://proempresa-api.onrender.com
```

Configuración Vercel:

- Framework: Vite
- Root Directory: `01_core_proempresa/frontend_react`
- Install: `npm install --legacy-peer-deps --no-audit --no-fund`
- Build: `npm run build`
- Output: `dist`
